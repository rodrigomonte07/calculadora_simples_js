# 🧮 Calculadora Simples em JavaScript

## 📌 Descrição

Este projeto consiste em uma calculadora simples desenvolvida em JavaScript, com o objetivo de praticar conceitos fundamentais da linguagem, como:

* Entrada de dados com `prompt()`
* Conversão de tipos (`Number`)
* Operadores aritméticos
* Operadores de atribuição
* Estruturas condicionais (`if/else`)
* Validação de dados

---

## 🚀 Funcionalidades

A aplicação permite ao usuário:

* Inserir dois números
* Realizar as seguintes operações:

  * Soma
  * Subtração
  * Multiplicação
  * Divisão
  * Resto da divisão (módulo)
* Visualizar o uso de operadores de atribuição:

  * `+=`
  * `-=`
  * `*=`
  * `/=`
  * `%=`
* Validar entradas inválidas (não numéricas)
* Evitar divisão por zero

---

## ▶️ Como Executar

1. Abra um navegador (Google Chrome, Edge, etc.)
2. Pressione `F12` para abrir o console
3. Cole o código JavaScript
4. Execute e insira os valores solicitados

---

## 💻 Exemplo de Uso

```
Digite o primeiro número: 10
Digite o segundo número: 2

Soma: 12
Subtração: 8
Multiplicação: 20
Divisão: 5
Resto: 0

Após += : 12
Após -= : 10
Após *= : 20
Após /= : 10
Após %= : 0
```

---

## ⚠️ Tratamento de Erros

O sistema verifica:

* Se os valores digitados são números válidos
* Se há tentativa de divisão por zero

Exemplo de erro:

```
Erro: digite apenas números válidos.
```
